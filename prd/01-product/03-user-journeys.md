# User Journeys

> Core user flows that define the Ordo Creator OS experience.

## Journey 1: First-Time Creator Setup

**Persona**: Sarah, 26, YouTube creator (5K subs), overwhelmed with inconsistent uploads.

```
1. SIGN UP
   Sarah lands on Ordo > Signs up with Google OAuth
   ↓
2. WORKSPACE CREATION
   Creates "Sarah's Channel" workspace > Selects FREE tier
   ↓
3. GUIDED ONBOARDING
   - Choose content type: Video (primary), Twitter (secondary)
   - Set publishing goals: 2 videos/week, 5 tweets/week
   - Import existing ideas (optional)
   ↓
4. FIRST IDEA CAPTURE
   Types her first idea: "React hooks mistakes beginners make"
   AI suggests 3 variations and validates the idea
   ↓
5. PIPELINE INTRODUCTION
   Promotes idea to pipeline > Sees Kanban board
   Content moves: Scripting > Filming > Editing > Published
   ↓
6. CONSISTENCY DASHBOARD
   Sees her Consistency Score (0%) and empty heatmap
   Goal: Turn that heatmap green
```

**Success Criteria**: Sarah creates her workspace and captures her first idea in < 3 minutes.

## Journey 2: Daily Creator Workflow (Power User)

**Persona**: Marco, 31, multi-platform creator (50K combined), uses Ordo daily.

### Morning (Mobile - 10 min)
```
1. MORNING RITUAL
   Opens mobile app > "Morning Review" ritual auto-starts
   ↓
2. QUICK METRICS CHECK
   Sees yesterday's stats: +127 subs, 2.3K views
   Consistency Score: 87% (12-day streak!)
   ↓
3. INBOX SCAN
   Reviews 8 new comments > Quick-replies to 3
   Stars 2 comments for content ideas
   ↓
4. CAPTURE IDEAS
   Voice-captures idea while commuting
   Telegram bot auto-captures a link shared by a friend
```

### Afternoon (Desktop - Studio Mode)
```
5. PIPELINE REVIEW
   Opens Kanban > 2 pieces in "Editing", 1 in "Scripting"
   ↓
6. SCRIPT WRITING (Creators Studio)
   Opens AI Studio > Writes video script
   AI suggests stronger hooks and retention points
   Timer running: 45-min focused session
   ↓
7. TITLE/THUMBNAIL OPTIMIZATION
   AI generates 5 title variations with CTR predictions
   Picks "Stop Making These 5 React Hooks Mistakes"
   ↓
8. SCHEDULE PUBLISHING
   Drags content to Thursday 10 AM on calendar
   AI confirms: "85% of your audience is active at this time"
```

### Evening (Mobile - 5 min)
```
9. EVENING CHECK
   Reviews daily progress: 1 piece scripted, 1 scheduled
   XP earned: 45 XP (+streak multiplier = 67 XP)
   ↓
10. PLAN TOMORROW
    Drags "Film B-roll" to tomorrow's calendar
    Sets filming reminder for 2 PM
```

**Success Criteria**: Marco completes his full content workflow across 3 platforms in under 1 hour total.

## Journey 3: Idea to Published Content

**Persona**: Any creator moving one piece from idea to publication.

```
CAPTURE (5 seconds)
│  Quick-capture: "Tutorial on CSS Grid vs Flexbox"
│  AI auto-tags: #tutorial #css #webdev
│
▼ VALIDATE (2 minutes)
│  View in Validation Board
│  Score: Impact 4/5, Effort 2/5
│  AI says: "High search volume, moderate competition. Go for it."
│
▼ TRANSFORM (1 minute)
│  Click "Transform to Content"
│  AI generates: Outline, 5 hooks, 3 thumbnail concepts, 20 hashtags
│  Review and customize > Accept
│
▼ SCRIPT (30-60 minutes)
│  Open in Creators Studio
│  Write script with AI assistance
│  AI flags: "Section 3 might lose retention. Consider cutting."
│
▼ PRODUCE (varies)
│  Move to "Filming" > Smart checklist appears:
│  [ ] B-roll shots list
│  [ ] Screen recording setup
│  [ ] Intro/outro assets from Series
│  Timer tracks production time
│
▼ EDIT (varies)
│  Move to "Editing"
│  Upload final file > Attach assets
│
▼ REVIEW (5 minutes)
│  Final check: Title, thumbnail, description, tags
│  AI optimizes SEO description
│  Team review (if applicable)
│
▼ SCHEDULE (1 minute)
│  Drag to calendar slot
│  Set cross-platform distribution:
│  - YouTube: Full video (Thursday 10 AM)
│  - Twitter: Thread summary (Thursday 11 AM)
│  - LinkedIn: Key takeaways (Thursday 12 PM)
│
▼ PUBLISHED
│  Auto-publishes at scheduled time
│  Analytics start tracking
│  Consistency Score updates
│  XP awarded: 100 XP
```

## Journey 4: Content Repurposing

**Persona**: Creator who published a YouTube video and wants to maximize reach.

```
1. VIDEO PUBLISHED
   "CSS Grid vs Flexbox" video goes live on YouTube
   ↓
2. OPEN REMIX TOOL
   Select the published content piece
   Click "Remix / Atomize"
   ↓
3. AI ANALYSIS
   AI analyzes transcript > Identifies:
   - 3 viral-worthy moments (high engagement potential)
   - 5 tweetable quotes
   - 2 newsletter-worthy sections
   ↓
4. GENERATE VARIANTS
   One-click generation:
   - Twitter thread (10 tweets summarizing key points)
   - LinkedIn post (professional framing)
   - Instagram carousel (5 slides with key visuals)
   - TikTok script (60-second hook from viral moment)
   - Newsletter section (deeper dive on one topic)
   ↓
5. REVIEW & CUSTOMIZE
   Edit each variant to match platform voice
   Adjust hashtags per platform
   ↓
6. SCHEDULE ALL
   Bulk schedule across platforms
   Staggered timing for maximum reach
   ↓
7. TRACK PERFORMANCE
   Unified analytics shows which platform
   performed best for this content
```

   Mark deliverables as completed
   Move to: DELIVERED

6. PAYMENT

## Journey 6: Team Collaboration

**Persona**: Creator with an editor and social media manager.

```
1. WORKSPACE SETUP
   Creator creates "Team Workspace" on PRO plan
   Invites: Editor (MEMBER role), Social Manager (MEMBER role)
   ↓
2. ROLE-BASED ACCESS
   Creator: Full access (OWNER)
   Editor: Can edit content, move pipeline stages
   Social Manager: Can schedule posts, manage inbox
   ↓
3. CONTENT HANDOFF
   Creator writes script > Moves to "Editing"
   Editor receives notification
   Editor picks up, reviews, requests changes
   ↓
4. PUBLISHING FLOW
   Social Manager reviews scheduled content
   Adjusts timing based on analytics
   Schedules cross-platform distribution
   ↓
5. AUDIT TRAIL
   All actions logged in Workspace Audit Log
   Creator can review who did what, when
```
